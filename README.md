### Usage

```js
yarn install
yarn start
```

执行完上述命令后，浏览器会自动打开并在 `localhost:8080` 上展示 `hello,cpreact`。~~也可以在 [沙盒](https://codesandbox.io/s/0xk1r2zmk0) 中使用~~
